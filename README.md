# Таймер для перерывов (Помодоро)

Современное десктопное приложение для организации рабочего времени по методу Помодоро на Python с возможностью сворачивания в трей и режимом строгого перерыва.

## Особенности
*   **Современный Flat-дизайн:** Тёмный интерфейс, скруглённые углы и круговой индикатор прогресса, реализованный с помощью CustomTkinter.
*   **Сворачивание в трей:** При закрытии приложения оно продолжает работать в фоновом режиме, отображая иконку в системном трее. Двойной клик по иконке открывает приложение обратно.
*   **Беззвучные уведомления:** Всплывающие сообщения о завершении таймера реализованы нативно для Windows без раздражающих звуковых сигналов.
*   **Режим "Строгого перерыва" (Strict Break):** Во время перерыва блокирует экран полупрозрачным окном поверх всех остальных приложений, не давая продолжить работу до окончания таймера.
*   **Сохранение настроек:** Длительность сессий и режим строгого перерыва сохраняются в файле `config.json` в рабочей директории.

## Быстрый запуск и компиляция
Процесс подготовки окружения и компиляции полностью автоматизирован.

1.  Убедитесь, что на компьютере установлен Python (версии 3.8 и выше).
2.  Дважды кликните по файлу **`setup.bat`**.
    *   Скрипт автоматически создаст виртуальное окружение `.venv`.
    *   Установит все зависимости (CustomTkinter, Pillow, pystray, plyer, PyInstaller).
    *   Сгенерирует файл иконки `icon.ico`.
    *   Скомпилирует проект в один файл **`timer.exe`**.
3.  После завершения работы скрипта, готовый файл появится в папке `dist/`.

## Запуск в режиме разработки
Если вы хотите запустить проект напрямую из исходного кода:

```bash
# Создание окружения и установка зависимостей
python -m venv .venv
call .venv\Scripts\activate
pip install -r requirements.txt

# Запуск приложения
python main.py
```

## Управление
*   **Работа (Focus) / Перерыв (Break):** Переключайте режимы с помощью верхних кнопок.
*   **Старт / Пауза:** Запускает и останавливает таймер.
*   **Сброс:** Сбрасывает текущий интервал на исходное значение.
*   **Системный трей:** Клик правой кнопкой мыши по иконке помидора в трее открывает контекстное меню (Открыть, Пауза/Старт, Выход).
